Joern is written in Scala. Please make sure a Java Virtual Machine
(version 8 or higher) is installed. Linux, MacOSX and FreeBSD are
supported. Joern has not been tested on Windows.

```bash
wget https://github.com/ShiftLeftSecurity/joern/releases/latest/download/joern-cli.zip
unzip joern-cli.zip
cd joern-cli
./joern
```
